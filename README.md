# res_pjsip_cisco_park
Asterisk PJSIP module for FreePBX 17 that enables the Park button on Cisco CP-8811, CP-8841, CP-8851, CP-8861 and other CP-8xxx IP phones. Intercepts proprietary x-cisco-remotecc ParkMonitor REFER requests, parks the call into an Asterisk parking lot, and displays the reserved parking slot on the phone screen.
